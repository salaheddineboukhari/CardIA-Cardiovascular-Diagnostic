# CardIA - Assistance Intelligente au Diagnostic Cardiovasculaire

CardIA est une application web conçue pour assister le personnel médical dans le diagnostic cardiovasculaire en exploitant la puissance du Deep Learning et de l'IA explicable (XAI).

## Fonctionnalités Clés
* **Classification ECG :** Réseau de neurones dédié à la classification des signaux ECG (ajusté sur la base MIT-BIH).
* **Prédiction du Risque :** Modèle d'évaluation et d'optimisation des risques cardiovasculaires (via Optuna).
* **IA Explicable (XAI) :** Intégration de modules d'explicabilité pour aider les cliniciens à interpréter les prédictions et les variables clés.
* **Génération de Rapports :** Dashboard d'analyse complet avec Flask et édition automatisée de synthèses de diagnostic.

## Stack Technique
* **Backend :** Python, Flask (Architecture MVC)
* **Intelligence Artificielle :** PyTorch, Scikit-learn, Optuna
* **Interface Utilisateur :** HTML5, CSS3, JavaScript
